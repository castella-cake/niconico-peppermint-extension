<ユーザーデータの管理

Import

エクスポートした設定をインポートします。
事前に現在のストレージをエクスポートしておくことをおすすめします。
ストレージをインポートする前に、悪意のあるコードが含まれていないか確認してください！
インポート結果はここに表示されます。

Export

現在の設定をクリップボードにコピーします。デバッグ目的としても使用できます。
これにはストックされたシリーズなどのデリケートな情報が含まれているので、誰かに共有する際は十分注意してください。
コピーした設定を保存する際は、json形式で保存することをおすすめします。
ストレージがありません。

Local storage

現在のローカルストレージをコピー/消去します。ストックしているシリーズのエピソードリストや、ニコレポをキャッシュするのに使用されます。
消去機能は基本的にデバッグ用ですが、エピソードリストやニコレポの動作がおかしい場合に役立つ可能性が僅かにあります。(同期ストレージに保存されている設定は削除されません)
現在のローカルストレージ使用容量: 0 Byte

Reset

PepperMintの設定、シリーズストックなどのユーザーデータ、キャッシュを全て削除します。
